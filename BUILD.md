# HOW TO BUILD

## library build
```
1.>cd libneo_wolf_ssl
2.write PLATFORM=[platform] in Makefile 
	default PLATFORM=gnu-armhf
	'gnu-armhf' is raspberry cpu 
	
3.>make
```

```
1.>cd console_neo_wolf_ssl
2.write PLATFORM=[platform] in Makefile

	'gnu-armhf' is raspberry cpu 
	
3.>make
```
